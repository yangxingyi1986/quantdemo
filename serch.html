getEquJY --股票基本信息

GET

/api/listedCorp/getEquJY.json 或 /api/listedCorp/getEquJY.csv
说明

获取股票的基本信息，包含股票交易代码及其简称、股票类型、上市状态、上市板块、上市日期等；上市状态为最新数据，不显示历史变动信息。
URL 用例 /api/listedCorp/getEquJY.json?field=&secID=&ticker=000001,000002,000006,000005,600000,600002,600003&equTypeCD=&listStatusCD=&listSectorCD=&exchangeCD=
请求和返回字段

请求字段：

名称
类型
可空
描述
exchangeCD string 三选一 股票交易市场，可选择：XSHG上海证券交易所，XSHE深圳证券交易所。
secID string 三选一 一只或多只证券代码，用,分隔，格式是“数字.交易所代码”，如000001.XSHE。如果为空，则为全部证券。
ticker string 三选一 一只或多只股票代码，用,分隔，如000001,000002。
equTypeCD string 是 股票类别，如A股，B股。
listSectorCD string 是 股票上市板块，可选择主板，中小板，创业板。
listStatusCD string 是 股票上市状态，可选择上市，暂停，退市等，默认为上市。
field string 是 可选参数，用,分隔,默认为空，返回全部字段，不选即为默认值。返回字段见下方
